# Lab 13 – Directed Acyclic Graph (DAG)

## AIM
To construct Directed Acyclic Graph (DAG) for detecting common subexpressions in expressions.

## THEORY
DAG is used in compiler optimization to eliminate redundant computations.

Common subexpressions are computed only once and reused later, improving execution efficiency.

## ALGORITHM
1. Read expressions sequentially.
2. Identify operators and operands.
3. Create nodes for each operation.
4. Reuse existing nodes for repeated expressions.
5. Display DAG structure.

## OUTPUT
Program constructs DAG and identifies shared subexpressions.

## CONCLUSION
DAG successfully optimized repeated computations.
